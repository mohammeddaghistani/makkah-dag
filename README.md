AI EXTENSION PACKAGE
====================

This package extends the Municipal Rent Valuation System with AI layers
WITHOUT modifying or removing any existing code or rules.

Included:
- Explainable AI (textual justification)
- Outlier detection (warnings only)
- Scenario comparison (what-if summaries)
- Strategic insights (management-only)

Usage:
Copy the `ai/` folder into your project root.
Optional integration.
